# Plot multiple feature effects

Plot multiple feature effects

## Usage

``` r
plot_partial_dependence_multi(
  pdp_list,
  title = "Feature Effects",
  colors = NULL,
  ncol = 2L
)
```

## Arguments

- pdp_list:

  Named list of feature effects.

- title:

  Overall title.

- colors:

  Line colors.

- ncol:

  Number of subplot columns.

## Value

A Plotly htmlwidget.
